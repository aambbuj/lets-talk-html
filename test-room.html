Video Room Test

This page helps test the video room functionality.

Note: Make sure your backend server is running before testing the video room.

1. Test Video Room

Click the button below to open the video room in a new tab:

Open Video Room

2. Test with Different Room Names

Test with various room configurations:

Room 1 (Alice) Room 2 (Bob) Meeting Room (John)

3. Test Video Call Features

Once in the video room, test these features:

  • Camera and microphone access
  • Mute/unmute button
  • Video on/off button
  • Screen sharing
  • Picture-in-Picture
  • Debug log (click the bug icon)
  • Call timer
  • Connection status

4. Test Multiple Users

To test video calls between users:

  1. Open the same room URL in two different browser tabs/windows
  2. Allow camera and microphone access in both
  3. Wait for the connection to establish
  4. You should see both local and remote video

5. Troubleshooting

If you encounter issues:

  • Check the browser console for error messages
  • Ensure your backend server is running
  • Check if camera/microphone permissions are granted
  • Verify the socket connection is established
  • Use the debug log in the video room
