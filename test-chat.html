Chat Room Test

This page helps test the chat room functionality.

Note: Make sure your backend server is running before testing the chat room.

1. Test Chat Room

Click the button below to open the chat room in a new tab:

Open Chat Room

2. Test with Different Room Names

Test with various room configurations:

Chat Room 1 (Alice & Bob) Chat Room 2 (John & Jane) Meeting Chat (User & Host)

3. Test Chat Features

Once in the chat room, test these features:

  • Message sending and receiving
  • Typing indicators
  • User join/leave notifications
  • Video call button (switches to video room)
  • Back button navigation
  • Real-time updates

4. Test Multiple Users

To test chat between users:

  1. Open the same chat room URL in two different browser tabs/windows
  2. Use different names for each tab
  3. Send messages from one tab to see them appear in the other
  4. Test typing indicators

5. Test Video Call Integration

Test the integration between chat and video:

  • Click the video call button in chat to switch to video room
  • Verify the room name and parameters are preserved
  • Test switching back to chat from video room

6. Troubleshooting

If you encounter issues:

  • Check the browser console for error messages
  • Ensure your backend server is running
  • Verify the socket connection is established
  • Check if the chat room JavaScript file is loading
  • Verify the CSS styles are applied correctly
